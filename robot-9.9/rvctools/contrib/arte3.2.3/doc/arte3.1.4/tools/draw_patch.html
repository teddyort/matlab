DESCRIPTION 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
   DRAW_PATCH(F, V, C, transparent)

   Draws a link using faces F and vertices V.
   C Defines the color of the robot.
   If transparent == 1, the robot is drawn with transparency.
